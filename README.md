# AVBotz

### Requirements

`python 3.6+` needed

Install required python packages:

```
    pip3 install opencv-python scipy numpy
```

### Usage

Question 1:
``` 
    python3 Q1_pathmarker.py <pathmarker.png>
```

Question 2:
```
    python3 Q2_align_to_target.py <align.in>
```

Question 3:
```
    python3 Q3_align_to_pinger.py <audio.in.txt>
```
